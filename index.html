📋 Hướng dẫn cập nhật ảnh nhanh nhất

1

Mở Google Drive folder Daily Wine

Vào thư mục ảnh sản phẩm, tìm đúng chai rượu cần cập nhật.

2

Lấy link ảnh

Click chuột phải vào file ảnh → Chia sẻ → Sao chép liên kết. Hoặc mở ảnh rồi copy URL trên thanh địa chỉ.

3

Dán vào ô bên trên

Chọn sản phẩm → dán link → nhấn Xem trước để kiểm tra → nhấn Lưu ảnh.

4

Ảnh lưu tự động ngay trên thiết bị này

Ảnh được lưu vào bộ nhớ trình duyệt (localStorage). Mỗi thiết bị/browser cần cập nhật riêng. Để đồng bộ nhiều máy, liên hệ kỹ thuật để embed vào code.
